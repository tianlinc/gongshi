## 用户需求

对工时系统软件（IEI Timer Faster）进行以下改动，**要求先做原型让用户审核，审核通过后再实施修改**。

### 1. 登录页 RDM 地址可配置 + 连通性检测

- 登录页增加 RDM 地址配置功能，用按钮触发弹窗配置（按钮不用太明显）
- 弹窗中可修改 RDM 地址，默认填充当前地址 `http://10.111.36.3:2029`
- 登录页自动校验 RDM 系统的连通性，网络能联通时在页面上显示绿色对勾
- RDM 地址需要全局配置，因为 URL 访问、RDM 跳转等都基于这个地址

### 2. 登录页样式改版

- 采用左右布局：左侧放置产品的简洁宣传语，右侧放置登录信息
- RDM 地址配置按钮的放置位置自行考虑
- 产品名称是 **IEI Timer Faster**，登录页标题不能叫"RDM工时系统"

### 3. 主页面头部布局优化

- 产品名称修正为 IEI Timer Faster
- 按钮分类整理（当前拥挤），按钮样式重新设计优化
- 第二行日期那一行充分利用起来（考虑任务列表是否放到日期选择那一行）
- 版本信息和升级功能归入【关于】，里面介绍产品信息
- License 相关信息归入 License 入口
- 整体页面的菜单、按钮重新设计优化

### 4. 发现新版本提示优化

- 右下角"发现新版本"提示里的"下载更新"按钮样式优化
- 点击提示后不需要独立的下载按钮，改为弹出版本信息，复用已有的升级逻辑
- 自动检测到新版本时，在【关于】按钮上加小红点让用户感知

### 5. 版本信息内容完善

- 版本信息区域需要展示当前版本的内容介绍（本版本更新了什么）
- 发现新版本时，也能看到新版本的介绍
