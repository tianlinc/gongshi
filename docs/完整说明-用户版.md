## 💡 使用流程

### 第一次使用

1. 双击 `start_v2.bat`
2. 等待启动完成
3. 浏览器自动打开 `http://localhost:5000`
4. 输入你的 RDM 用户名和密码
5. 点击"登录"

### 填写工时

1. 登录后看到工时填写界面
2. 点击"添加任务"
3. 输入任务名称
4. 填写每天的工作小时数
5. 设置完成进度
6. 点击"预览"
7. 确认后点击"提交工时"

### 快速操作

- **工作日8小时** - 一键填充周一到周五各8小时
- **保存草稿** - 保存当前填写的内容
- **加载草稿** - 恢复上次保存的内容
- **清空重填** - 清除所有内容重新填写

## 🛠️ 故障排查

### 问题：无法启动服务器

**解决**：
```bash
# 检查 Python 版本
python --version

# 如果版本太低，安装 Python 3.8+
# 下载地址：https://www.python.org/downloads/

# 手动安装依赖
pip install flask flask-cors requests pycryptodome beautifulsoup4
```

### 问题：登录失败

**解决**：
1. 确认能否访问 `http://10.111.36.3:2029`
2. 确认用户名密码是否正确
3. 尝试在浏览器中手动登录
4. 查看 Python 控制台的错误信息

### 问题：浏览器无法访问

**解决**：
1. 确认服务器已启动
2. 尝试访问 `http://127.0.0.1:5000`
3. 检查防火墙设置
4. 确认端口 5000 未被占用
